 𝐑𝐞𝐬𝐞𝐚𝐫𝐜𝐡 𝐐𝐮𝐞𝐬𝐭𝐢𝐨𝐧

Is Ethereum’s activity driven more by participation or engagement?

---

 𝐌𝐞𝐭𝐫𝐢𝐜𝐬 𝐔𝐬𝐞𝐝

- Total Transactions → overall network activity  
- Logic Transactions → smart contract engagement  
- Value Transactions → ETH transfer activity  
- Unique Senders → participation proxy  

---

 𝐌𝐞𝐭𝐡𝐨𝐝𝐨𝐥𝐨𝐠𝐲

Data was sourced from the Ethereum transactions table on Dune Analytics.

The analysis was conducted over a 14-day observation window using daily aggregated transaction data.

---

 𝗢𝗯𝘀𝗲𝗿𝘃𝗮𝘁𝗶𝗼𝗻𝘀

- Logic transactions remained consistently higher than value transactions.
- Both logic and value transactions showed a declining trend over time.
- Unique sender activity fluctuated and showed no clear evidence of sustained growth.

---

𝐈𝐧𝐭𝐞𝐫𝐩𝐫𝐞𝐭𝐚𝐭𝐢𝐨𝐧

The observed activity suggests that Ethereum usage during this period was driven more by engagement through smart contract interactions than by participation growth.

---

𝐋𝐢𝐦𝐢𝐭𝐚𝐭𝐢𝐨𝐧𝐬

- The analysis is limited to a 14-day window and may not reflect long-term trends.
- Unique addresses are only a proxy for users.
- Elevated logic transaction activity may include bot or automated contract interactions.

---

 𝐂𝐨𝐧𝐜𝐥𝐮𝐬𝐢𝐨𝐧

Based on the observed period, Ethereum’s activity appears to be driven more by engagement than participation, as logic transactions consistently exceeded value transactions while participation metrics remained inconsistent.
